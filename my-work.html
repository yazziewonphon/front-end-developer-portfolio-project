Weather App

The weather application was built using Bootstrap and API integration that returns real-time data from any city in the world. I learned how to develop and implement advanced coding techniques from the SheCodes Plus Workshop with instructor Matt Delac, and I was able to successfully create a lite simulated version of the Google weather application. By simply inputing a city in the search engine or allowing the app to geolocate your current position, the app will show the user the current temperature, humidity, wind speed, weather condition, and time.

AJAX, Open Weather API Integration, Geolocation API, Bootstrap, Javascript, Git, Github, Web Hosting (Netlify), and VS Code.

Sen Laptop

The Sen Laptop homepage was my first project on static site development when I went through SheCodes Basics with instructor Matt Delac. This landing page focuses on using basic coding knowledge such as HTML and CSS to practice making a structure and then designing it. As a former print designer, I wanted my page to mimic 90's magazines of the tech industry which is now a far cry from the power of white space in today's UI. Despite its limited usability, this project reveals traces of my signature style as a front-end developer.

VS Code and Figma. Hosted on SheCodes.
